## Design System

The app theme now follows the Stitch `Velocity Dark` direction:

- Dark navy/slate surfaces: `#0b1326`, `#131b2e`, `#171f33`, `#222a3d`
- Electric cyan primary: `#00e5ff`
- Inter for primary UI text
- Space Grotesk for metadata, labels, and compact technical text
- Low-radius, high-density panels with 1px borders instead of heavy shadows
